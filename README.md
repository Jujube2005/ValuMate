# ValuMate

## Features
- Add product / quantity / price
- Categorize transactions (Income / Expense)
- Compare product prices and choose the best product based on quantity and price.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, TypeScript
